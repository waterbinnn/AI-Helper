- [OpenAI](https://platform.openai.com/docs/api-reference/chat/create)

### 기능

- 채팅 : 일상적인 대화, 구글 검색 후 답변 도출
- 제목 생성 : 블로그 게시물 또는 기타 콘텐츠에 대해 창의적이고 관련성 있는 제목을 생성
- 스크립트 생성 : 프레젠테이션, 비디오 등에 대한 스크립팅 가능 , 구글 검색 후 결과 도출
